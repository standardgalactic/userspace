# Mereological-Space-Ontology

An address and an ingredient list

## Motivation and Setup

The MSO begins from a simple premise: any world-description that treats “things” as stable must implicitly choose a decomposition of the world into parts, and must silently decide which compositions count as units. Mereology provides a language for this choice.

The MSO adds two crucial refinements. First, scale is not treated as an afterthought but as a primary index that organizes admissible parthood relations. Second, salient regime transitions are encoded through a glyph-string that can be read both as an ontological ladder of composition and as an epistemic ladder of exploration.
